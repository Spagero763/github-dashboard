# GitHub Activity Dashboard

A **frontend-only dashboard** built with **Next.js** that allows users to view GitHub profiles, repositories, and contribution stats in a visually appealing interface.

---

## **Features**

- Fetch GitHub user profile data:
  - Avatar
  - Username with link to profile
  - Bio and location
  - Followers / following count
  - Total stars and forks across all repos

- Display list of user repositories:
  - Star and fork counts
  - Repository description
  - Programming language
  - Filter repos by name
  - Sort by stars, forks, or name

- Show **Top 5 repositories by stars** in an interactive bar chart
- Loading skeleton for better user experience
- Rate limit error handling
- Keyboard shortcut support (Enter to search)
- Fully **responsive and mobile-friendly**
- Clean, modern UI built with **Tailwind CSS**
- Frontend-only; no backend required
