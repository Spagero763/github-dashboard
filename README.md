# GitHub Activity Dashboard

A **frontend-only dashboard** built with **Next.js 13** that allows users to view GitHub profiles, repositories, and contribution stats in a visually appealing interface.

---

## **Features**

- Fetch GitHub user profile data:
  - Avatar 
  -username 
  -followers public repositories
- Display list of user repositories with star counts
- Show **Top 5 repositories by stars** in an interactive bar chart
- Fully **responsive and mobile-friendly**
- Clean, modern UI built with **Tailwind CSS**
- Frontend-only; no backend required
